# Portfolio Website - Ahmad Sabryan S.Pd

Website portfolio profesional dengan desain UI/UX modern menggunakan vanilla CSS.

## 🚀 Cara Menggunakan

### Membuka Website
1. Buka file `index.html` dengan browser (Chrome, Firefox, Edge)
2. Atau double-click file `index.html`

### Mengganti Background dengan Foto Anda
1. Siapkan foto Anda (JPG/PNG, minimal 1920x1080px)
2. Rename foto menjadi `background.jpg`
3. Replace file `background.jpg` yang ada di folder ini
4. Refresh browser

## 📁 Struktur File

```
Web Portofolio/
├── index.html      # File HTML utama
├── style.css       # File CSS dengan design system
├── background.jpg  # Background image (ganti dengan foto Anda)
└── README.md       # File ini
```

## ✏️ Customization

### Mengganti Nama & Informasi
Edit file `index.html`:
- **Line 25**: Nama di navbar
- **Line 62**: Nama di hero section
- **Line 88-96**: Deskripsi about section
- **Line 195-210**: Social media links

### Mengganti Warna Tema
Edit file `style.css` bagian `:root` (line 8-18):
```css
--color-primary: #3b82f6;    /* Warna utama */
--color-secondary: #8b5cf6;  /* Warna sekunder */
```
